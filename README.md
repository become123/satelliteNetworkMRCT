
### 執行指令
    
- 編譯:
    ```bash
    make clean
    make
    ```

- 執行:
    ```bash
    ./satelliteNetworkMRCT.exe 7P_16Sats_edgeWeight.txt 
    ./satelliteNetworkMRCT.exe 6P_22Sats_edgeWeight.txt 
    ```
    - 依照想要執行的星群拓譜檔案，選擇其中一個執行
        - 7P_16Sats_edgeWeight.txt : 紀錄7P_16Sats星群拓譜的檔案
        - 6P_22Sats_edgeWeight.txt : 紀錄6P_22Sats星群拓譜的檔案
